关于 WMU

WMU是为移动端提供的基础 UI 框架。目的是为了手机H5页面提供一个常用的组件库，减少重复工作。

{% include ad.html %}

不重复造轮子

我们并没有重复造轮子。
主要借鉴 ratchet 的形态并以此为基础，整合和吸收了很多优秀的前端框架，如:Ratchet、framework7、Amaze等……，精简体积，此外还定制增强了一些工具类开源库。

兼容性

兼容到 iOS 6+ 以及 Android 4.0+

如发现有bug请前往github上给我们提 issue,或者直接联系我们（联系方式在下面）。
